Mac使用疑点Q&A

2022-08-02
  • Markdown不支持段首空格的展示？
    option+space输入两个全角空格，宽度整整两个汉字很整齐
  • 直接在 Markdown 里用空格和 Tab 键缩进在渲染后会被忽略掉？
    借助 HTML 转义字符在行首添加空格来实现，&ensp;代表半角空格，&emsp; 代表全角空格。
  • Markdown制作表单时内容物需要用到｜？
    借助 HTML 转义字符&#124;
  • 更改中/英切换键为escape后怎么切换中英文？
    control+space
